# event-registration-system# Event Registration System

This is a simple web project created using HTML, CSS and JavaScript.

## Problem Statement
Create a webpage where users can register for an event by entering their details.

## Technologies Used
- HTML
- CSS
- JavaScript
- VS Code

## Project Structure

event-registration-system
│
├── index.html
├── success.html
└── README.md

## Steps to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double click **index.html**.
4. Fill the registration form.
5. Click Register to see success page.
